Overview

This project presents a Thermal-Aware Adaptive Communication System for wearable healthcare devices using Body Area Network (BAN) technology. The system continuously monitors physiological parameters such as heart rate, body temperature, and blood pressure through wearable sensor nodes.

Traditional BAN systems use static communication methods that often lead to excessive heat generation, higher energy consumption, communication delays, and reduced device lifetime. To overcome these limitations, this project introduces a temperature-aware adaptive routing mechanism that dynamically controls node participation based on temperature, activity level, and communication efficiency.

The proposed system improves:

Thermal management
Energy efficiency
Communication reliability
Network stability
Communication delay reduction

System Architecture

Network Generation

The system generates wearable sensor networks dynamically and visualizes node communication paths for analysis.

Thermal-Aware Routing

The routing mechanism selects optimal nodes for communication while reducing the activity of overheated nodes.

Performance Analysis

The system compares temperature rise, energy consumption, and communication delay between existing and proposed approaches.
